## How to Customize for Your Business

### Step 1: Update Basic Information

1. **Replace "Your Business Name" / "Your Company Name"** throughout the HTML file
2. **Update email addresses** (contact@yourdomain.com)
3. **Update phone numbers** (+1-XXX-XXX-XXXX)
4. **Replace domain URLs** (https://yourdomain.com)
5. **Update location information** (address, city, coordinates)

### Step 2: Customize Content

1. **Update H1 tag** - Should contain your main keyword
2. **Update meta description** - 150-160 characters, include keywords
3. **Update page content** - Make it relevant to your business
4. **Add your services/products** - Update service cards
5. **Update FAQ section** - Add questions relevant to your business

### Step 3: Add Images

1. Create and add:
   - `og-image.jpg` (1200x630px) - For social media sharing
   - `twitter-image.jpg` - For Twitter cards
   - `logo.png` - Your business logo
   - `favicon.ico` - Website favicon
   - `apple-touch-icon.png` - For iOS devices

### Step 4: Update Structured Data

1. **Organization schema** - Update with your business details
2. **LocalBusiness schema** - Add your address and hours
3. **Service/Product schema** - Update with your offerings
4. **FAQPage schema** - Ensure questions match your FAQ section

## SEO Best Practices for Google Rankings

### 1. Keyword Research
- Research relevant keywords for your business
- Use long-tail keywords (3-4 words)
- Include keywords naturally in content
- Use keywords in H1, H2, and first paragraph

### 2. Content Optimization
- Write high-quality, original content
- Use proper heading hierarchy
- Include keywords naturally (don't keyword stuff)
- Write for humans first, search engines second
- Aim for 1000+ words of quality content

### 3. Technical SEO
- Ensure fast loading times (< 3 seconds)
- Make site mobile-friendly
- Use HTTPS (SSL certificate)
- Create XML sitemap
- Create robots.txt file
- Fix broken links
- Optimize images (compress, use alt tags)

### 4. AEO Optimization
- Create FAQ sections with direct answers
- Use question-answer format
- Target featured snippet keywords
- Optimize for voice search queries
- Use structured data for FAQs

### 5. LLM Optimization
- Use clear, semantic HTML
- Structure content logically
- Use proper HTML5 semantic elements
- Include context and definitions
- Link related content

## Additional Files to Create

### robots.txt
```
User-agent: *
Allow: /

Sitemap: https://yourdomain.com/sitemap.xml
```

### sitemap.xml
Create an XML sitemap listing all your pages. Update the date regularly.

### .htaccess (if using Apache)
For redirects, caching, and security headers.
